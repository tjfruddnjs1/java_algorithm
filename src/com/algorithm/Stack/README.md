# Stack

## Stack Class 정리

1. public Element push(Element item)
- 데이터 추가

2. public Element pop();
- 최근 추가한 데이터(Top) 삭제

3. public Element peek();
- 최근 추가한 데이터(Top) 조회

4. public boolean isEmpty();
- stack 값이 비었는지 확인

5. public int search(Object o);
- 인자 값으로 받은 데이터 위치 반환
